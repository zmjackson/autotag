# autotag

![screenshot](/pictures/autotag2.png)

autotag is a 3D lidar/point cloud and classification viewer built in C++ Qt OpenGL. Right now it supports lidar data in the form of .ply files and tracking/classification data in the form of JSON files used in Argo AI's argoverse. Support for different formats is one of the many planned features.

- [tinyply](https://github.com/ddiakopoulos/tinyply)
- [jsoncpp](https://github.com/open-source-parsers/jsoncpp)
- [BreezeStyleSheets](https://github.com/Alexhuszagh/BreezeStyleSheets)
